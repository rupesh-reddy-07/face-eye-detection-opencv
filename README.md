# Real-Time Face & Eye Detection with OpenCV  

A Python project that detects faces and eyes in real-time using OpenCV and Haar cascades.

## Features  
✔ Real-time face detection  
✔ Eye detection within detected faces  
✔ Lightweight and fast (uses Haar cascades)  
✔ Works with webcam or video files  

## Technologies Used  
- **Python 3**  
- **OpenCV (cv2)**  
- **Haar Cascade Classifiers**
